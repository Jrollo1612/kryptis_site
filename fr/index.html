Traducteur Morse

Bienvenue sur le Traducteur Morse!

C'est un outil simple pour traduire le texte en code Morse et vice-versa.

Quand vous êtes sur l'application, écrivez votre texte dans le panneau de droite, et il sera automatiquement traduit en code Morse. Si vous voulez taper en Morse, écrivez "/" et un clavier de 4 symboles apparaîtra. Ensuite, cliquez sur eux pour écrire en Morse.
