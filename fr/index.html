Traducteur Morse

À propos de l'application

Bienvenue sur Traducteur Morse.

Cet outil traduit le texte en code Morse et le code Morse en texte.

Dans l'application, écrivez votre texte dans le panneau de droite pour une traduction automatique. Pour écrire en Morse, tapez "/" pour afficher un clavier Morse à 4 symboles.
