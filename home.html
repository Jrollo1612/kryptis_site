About the app

Morse Translator is a simple desktop application that allows you to translate text to Morse code and vice versa.

Go to downloads See reviews

The morse code is a method of transmitting text information as a series of tones, short and long.
It was invented by Samuel Morse in the 1830s, to transmit messages over telegraph lines.

Then, morse code became widely used in radio communication and maritime distress signals.
Currently, morse code is still used in Scouts activities. I did this translator firstly because it was a necessary project to have the Informatic scout badge.
