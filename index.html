Morse Translator

Welcome to the Morse Translator!

This is a simple tool to translate text into Morse code and vice versa.

When you are on the app, write your text in the right panel, and it will automatically translate it to Morse code. If you want to type in morse, write "/" and a keyboard of 4 symbols will appear. Then, click on them to write in morse.
